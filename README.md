# TxT - Twitter Clone

Full-stack Twitter clone built with React + Vite (frontend) and Express + SQLite (backend).

## Features

- Sign up / Login with JWT auth
- Create, like, retweet, and reply to posts
- User profiles with bio, avatar, and follow system
- Notifications
- Brutalist black & white design

## Run Locally

```bash
npm install
npm run server   # Terminal 1 - Express on :3001
npm run client   # Terminal 2 - Vite on :5173
```

Open http://localhost:5173
